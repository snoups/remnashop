#### Аппаратные

- **ОС**: рекомендуется **Ubuntu** или **Debian**
- **RAM**: минимум 2 ГБ, рекомендуется 4 ГБ
- **CPU**: минимум 2 ядра, рекомендуется 4 ядра
- **Хранилище**: 20 ГБ
  <Tip>
    Remnashop может работать с более низкими характеристиками, однако это не рекомендуется. Некоторые фоновые процессы могут быть ресурсоёмкими.
  </Tip>

---

#### Программное обеспечение

Remnashop требует наличия Docker с плагином Docker Compose.

<Accordion title="Установка Docker с помощью официального скрипта" defaultOpen>
  ```text
    sudo curl -fsSL https://get.docker.com | sh
  ```
</Accordion>
