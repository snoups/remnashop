Telegram поддерживает кастомные (премиум) эмодзи в кнопках и сообщениях. Чтобы их использовать, у бота должен быть платный username (Fragment) **или** у владельца бота должна быть активная подписка Telegram Premium.

## Как получить ID эмодзи

1. Отправьте нужный кастомный эмодзи боту [@getidsbot](https://t.me/getidsbot) или `@RawDataBot`.
2. В ответе бот вернёт данные сообщения, включая поле `custom_emoji_id`.

## Синтаксис

Прямо в значениях ключей `.ftl` поддерживаются два формата.

### Полный формат

```ftl
btn-example = <tg-emoji emoji-id="5406756500108501710">🎁</tg-emoji> Текст кнопки
```

### Короткий формат

```ftl
btn-example = <e id="5406756500108501710">🎁</e> Текст кнопки
```

Текст внутри тега (обычный эмодзи) — это запасной вариант, который показывается на клиентах без поддержки кастомных эмодзи. Рекомендуется всегда его указывать.
