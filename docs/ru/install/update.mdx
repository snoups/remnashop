---
title: "Обновление"
description: "Как обновить Remnashop до последней версии"
---

Для обновления и перезапуска бота выполните следующую команду:

<CodeGroup>
    ```bash Обновление бота
    cd /opt/remnashop && docker compose pull && docker compose down && docker compose up -d && docker compose logs -f
    ```
</CodeGroup>

Встроенные ассеты (переводы и баннеры по умолчанию) поставляются внутри Docker-образа и обновляются автоматически вместе с ботом — сбрасывать их вручную больше не нужно.

При запуске после обновления бот:

- Подхватывает актуальные встроенные переводы и баннеры из образа.
- Сохраняет ваши пользовательские переводы (`custom.ftl`) и баннеры без изменений.
- При обновлении с версии ниже 0.8 переносит старые файлы переводов в папку `.legacy/` — в пользовательском томе остаётся только `custom.ftl`.

<Note>
    Ваши переопределения в `custom.ftl` и собственные баннеры в `assets/` никогда не затрагиваются обновлением. Подробнее — на странице [Ассеты](/docs/ru/install/assets).
</Note>
