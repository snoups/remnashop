#### 0.8.0 (19 июня 2026)

Новое:
- Добавлена система **промокодов** с поддержкой различных типов скидок и условий использования
- Добавлено уведомление активации промокода
- Добавлена **детальная статистика** по промокодам и их активациям
- Добавлена система **рекламных ссылок**: конфигуратор, учёт выручки и т.п.
- Добавлено отображение рекламной ссылки в уведомлении регистрации пользователя
- Добавлен **чёрный список** с массовой блокировкой пользователей, а также авто-обновляемые списки
- Добавлено уведомление о попытке регистрации пользователя из чёрного списка
- Добавлена **система бэкапов** базы данных и ассетов с уведомлениями
- Добавлены доп. настройки: выключать возможность удаления устройств для пользователей, перевыпуск подписки, сброс реф. ссылки и т.п.
- Добавлен раздел **«Доп. настройки»** в дашборде с настройками, не относящимися напрямую к другим разделам
- Переработана **система переводов**: пользовательские переводы имеют приоритет над встроенными
- Добавлена поддержка **10 новых локалей** (BG, DA, EL, FI, HU, NO, SV, TG, TH, ZH)
- Добавлена поддержка **кастомных эмодзи**
- Добавлен платёжный шлюз **Valutix** (теперь поддерживается 14 шлюзов)
- Улучшена система уведомлений: возможность направлять системные уведомления в **группу/топик**
- Добавлены события **torrent blocker** и «пользователь не подключился»
- Добавлены новые события об ошибках и исключениях
- Добавлено уведомление об отключённом инлайн-режиме при запуске
- Добавлена настройка **отображаемого имени** платёжного шлюза в настройках
- Добавлена поддержка оплаты через **Platega** без выбора метода
- Добавлен новый тип **сообщения** для доп. кнопок главного меню
- Добавлена настройка **цвета** доп. кнопок главного меню
- Добавлен флаг видимости **«только для подписчиков»** в редакторе доп. кнопок главного меню
- Добавлена резервная кнопка для подключения через веб-браузер (на случай проблем с MiniApp)
- Добавлена возможность задать **платный пробный период**
- Добавлена возможность продлить подписку на **изменённый план** (с уведомлением, что условия плана изменились)
- При удалении устройства теперь **сбрасываются активные соединения**
- Улучшена статистика **дохода за прошлый месяц**
- Добавлена кнопка возврата к списку рефералов (в окне профиля реферала)
- Добавлена возможность сбрасывать ссылку приглашения у пользователей
- Добавлена возможность **авто-отключение пробной подписки** при выходе из канала/группы
- Улучшена информативность логов в событиях ошибок и исключений
- Усилена защита шлюзов и вебхуков, повышена идемпотентность транзакций
- Проведён аудит безопасности, устранены уязвимости и мелкие баги

Исправления:
- Исправлен перевод с отображением скидки при покупке
- Исправлено снятие роли OWNER с предыдущего пользователя при смене BOT_OWNER_ID
- Исправлена валидация вебхука в YooMoney (обновление API)
- Покупка платного пробного плана исключена из реферальных наград
- Скидки больше не применяются к цене пробного периода
- Исправлен сбой при оформлении заказа со скидкой **100%**
- Исправлено сохранение типа покупки при оформлении подписки
- Исправлено отслеживание активности пользователей (список последних взаимодействующих пользователей)
- Исправлено сжатие архивных логов

Будущий Веб и API:
- Добавлен публичный **REST API** (планы, подписка, рефералы, авторизация)
- Добавлена авторизация через **JWT и cookie**, refresh-токены и вход через Telegram
- Добавлены **health-эндпоинты** и переключатель Swagger UI (`APP_SWAGGER_ENABLED`)

#### 0.7.4 (7 апреля 2026)

Новое:
- Добавлена поддержка **Remnawave 2.7.x**
- Добавлена стратегия сброса трафика **MONTH_ROLLING**: трафик сбрасывается по дате создания подписки, а не по фиксированному дню месяца
- Добавлена опциональная поддержка **SOCKS5 прокси** для доступа к Telegram API (настраивается через `BOT_PROXY_URL` в env)
- Добавлена **проверка совместимости версии Remnawave**: бот предупреждает, если версия подключённой панели устарела или не поддерживается
- Добавлено управление **скидкой на покупку**: отдельные личная и одноразовая скидка на пользователя, применяется наибольшая
- Добавлена кнопка переключения **доступности пробного периода** в редакторе пользователя
- Добавлен **индикатор типа скидки** в диалоге подписки: показывает, личная скидка или одноразовая
- Увеличена максимальная длина названий кнопок меню и названий планов с **16 до 32 символов**
- Улучшено создание платежа в YooKassa: добавлена логика повторных попыток при ошибке соединения
- Добавлены проверки прав при изменении роли пользователя: запрет на понижение собственной роли, защита роли Владельца и соблюдение иерархии ролей

Исправления:
- Исправлено сравнение ролей в списке администраторов: доступность редактирования теперь корректно рассчитывается относительно роли актора
- Исправлено условие отображения кнопки удаления роли в диалоге Администраторы
- Исправлена статистика подписок: учитываются только активные (текущие) подписки пользователей
- Исправлена статистика пробных периодов: теперь считаются уникальные пользователи, а не общее число пробных подписок
- Исправлен расчёт скидки: применяется максимальная из личной и одноразовой скидок
- Исправлен сброс покупочной скидки после успешной оплаты
- Исправлено форматирование сумм в платежах
- Исправлен вызов `delete_commands` при отключённом `setup_commands`
- Исправлено некорректное срабатывание уведомления о доступном обновлении
- Исправлена конфигурация подключения к Redis
- Исправлен переводчик: пустые строки после удаления опциональных плейсхолдеров больше не остаются
- История транзакций пользователя теперь отсортирована по дате создания (сначала новые)
