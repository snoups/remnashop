<Warning>
    Remnashop и его компоненты не поддерживают работу на суб-пути.
    (например, <Badge color="yellow">location /remnashop</Badge>, <Badge color="yellow">location /remnawave</Badge>)
</Warning>

<Note>
    **SSH Port Forwarding** не поддерживается. Используйте только один из обратных прокси ниже.
</Note>

<Tip>
    Для продолжения вам необходимо иметь зарегистрированное доменное имя.
</Tip>

<Tip>
    Обратный прокси должен указывать на контейнер с ботом по внутреннему адресу `remnashop:5000` — это Docker-имя сервиса из `docker-compose.yml`, доступное только внутри сети контейнеров.

    Маршрут: `https://your-domain/api/v1` → `http://remnashop:5000`
</Tip>
